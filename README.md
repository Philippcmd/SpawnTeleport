## Description
SpawnTP is a lightweight paper plugin to allow players teleporting to the worldspawn. In order to use the plugin just download it and put the `.jar`-file into your server's `plugins` folder and restart. Then players should be able to use the `/spawn` right away. No configuration to set the spawn is needed.
